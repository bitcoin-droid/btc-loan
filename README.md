# Bitcoin Loan Platform

A Bitcoin-backed lending platform allowing users to obtain short-term
USD loans using BTC collateral.

## Features
- Loan calculator (10% collateral)
- Secure application flow
- Bitcoin collateral instructions
- Manual approval (Phase 1)

## Technology
- Frontend: HTML, CSS, JavaScript
- Hosting: GitHub Pages

## Roadmap
- Backend API (Node.js)
- Database (PostgreSQL)
- Bitcoin payment gateway integration
- KYC & Compliance
- Admin dashboard

## Disclaimer
This platform is under active development.
No funds are accepted without manual verification.
